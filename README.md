Fine & Fragrance - Premium E-Commerce Platform
A sophisticated, full-stack e-commerce solution for a luxury fragrance boutique. This project demonstrates a complete shopping lifecycle, from secure user authentication to a robust administrative inventory system.

✨ Core Features
🛍️ For Customers
Secure Authentication: User registration and login with encrypted passwords.

Dynamic Catalog: Browse a curated collection of fragrances with detailed descriptions and pricing.

Smart Shopping Cart: Add, update, or remove items seamlessly before proceeding to checkout.

Checkout Workflow: A structured multi-step process ready for payment gateway integration.

🛡️ For Administrators
Secure Dashboard: A private area to manage the entire store operations.

Inventory Management (CRUD): Full Create, Read, Update, and Delete capabilities for products and categories.

Order Monitoring: Track customer orders and status in real-time.

🛠️ Technology Stack
Frontend: HTML5, CSS3, JavaScript (Vanilla)

Backend: Core PHP

Database: MySQL

Server: Apache (XAMPP/WAMP)

Integration Ready: Configured for Razorpay payment gateway integration.

📂 Project Structure
Plaintext
├── assets/             # CSS styling, JS scripts, and product images
├── classes/            # PHP classes for database and user logic
├── admin/              # Admin panel files and inventory CRUD
├── includes/           # Header, Footer, and Navigation fragments
├── config/             # Database connection strings
└── index.php           # Storefront landing page
🚀 Getting Started
Clone the Repo:

Bash
git clone https://github.com/PriteshBD/Fine-and-Fragrance.git
Database Configuration:

Import the database.sql file via phpMyAdmin.

Update config/db.php with your local MySQL credentials.

Local Hosting:

Place the project folder in your htdocs directory.

Open http://localhost/Fine-and-Fragrance in your browser.
